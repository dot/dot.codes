定期的に罹患するblogを作りたい病になったので、触ってみたかったAstroとbunを使ってブログを作ってみる。
テンプレートは、Astroの[Themes](https://astro.build/themes/)なかから[astro-paper](https://github.com/satnaing/astro-paper)を選んだ。わりとメンテされてそうで、デザインがシンプルで改修しやすそうなものを選んだ。

```
 rtx use -g bun@latest
```

で、bunを入れて、適当な場所で

```
bunx create-astro@latest -- --template satnaing/astro-paper
```

としてあとは指示に従って。`bun start`でローカルサーバが立ち上がるので、ちょこちょこと手を入れて終わり。

mdxも使ってみたかったので

```
bun astro add mdx
```

で、mdxが入る。あとは`content/blog`配下にmdxの拡張子でファイル作るとReactのコード埋め込みできるところまでは確認した。

[Netlify](https://www.netlify.com/)を使って、このブログを公開してみる。

よく分からないまま、astroの Github リポジトリをNetlify連携してみたら、masterにpushするだけで公開されるっぽい。便利だ。
ドメインとの連携はまだちょっと置いておいて、しばらくこのまま運用してみる。

あとやること

- Aboutのページを修正
- トップページのヘッダ部を修正
- デザイン(色味)を替えてみたい

が終わったら、dot.codesのドメインとして公開してみよ。
